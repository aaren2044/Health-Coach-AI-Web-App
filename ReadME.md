## Technology Stack

**Frontend:**  
- React + TypeScript + Vite  
- Tailwind CSS + Framer Motion  
- Chart.js / Recharts

**Backend:**  
- Node.js / Express  
- Supabase (Auth)  
- MongoDB

**Integrations:**  
- Apple HealthKit  
- Google Fit  
- Telegram Bot API
